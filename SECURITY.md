# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch state.

## Reporting a Vulnerability

Please report vulnerabilities privately to:

- `security@delta-sync.dev`

Include:

- affected component
- impact assessment
- reproduction details
- any proof-of-concept data

You should receive an acknowledgment within 72 hours.

## Disclosure

- Do not disclose publicly before a fix is available.
- Coordinated disclosure is preferred.
- We will credit reporters unless anonymity is requested.
